**2) Приложение AI-секретарь для протоколирования совещаний**
 
    APP__ai-meeting-secretary

    AI-секретарь автоматически:
	  · принимает аудио, видео, ссылки и текстовые сообщения;
	  · выполнять транскрибацию;
	  · анализирует содержание встречи; 
	  · формирует структурированный протокол в формате PDF (и в формате маркдауна .md);
	  · позволяет задавать вопросы по результатам встреч в режиме чата и отвечает на них.
	  
    Реализовано  в сервисе Google AI Studio.

**4) Приложение ENERGY (Fitness AI-Consultant)**
    
	APP__Fitness-AI-Consultant

	AI-Consultant строит диалог с потенциальным клиентом фитнес-центра «ENERGY» для взрослых

	· провести тёплый экспертный диалог,

	· привести к оформлению заявки 
	· (сбор контактов).

	Реализовано  в сервисе Google AI Studio.

**6) Приложение «АИ-Ветеринар»**

	APP__AI-veterinary-assistant

    Веб-приложение «АИ-Ветеринар»:
	· разговаривает с владельцем питомца
	· не ставит диагнозы
	· не назначает лекарства
	· сначала определяет, насколько ситуация срочная
	· затем спокойно и последовательно задаёт вопросы
	· в конце даёт чёткий и понятный план действий
	· всегда направляет к ветеринару, если есть риск для здоровья

	Приложение создано в сервисе Google AI Studio.


**7) Приложение «AI-сервис создания сказок»**

	- APP__fairy_tale_creation_service

    Веб-приложение создания интерактивных сказок для детей: 
	· родители задают ценность + выбирают мир/героя/антагониста (по 3 варианта с картинками), 
	· затем ребёнок читает главы и выбирает продолжение (2–4 варианта). 
	· Сказка ветвится и может продолжаться бесконечно (дерево глав), 
	· хранится локально в браузере.
